# 2.0 THE SYSTEM

## Typography as Structure
The identity is anchored by two typeface roles:
- **The Voice:** A high-contrast serif (editorial, aspirational)
- **The Data:** A utilitarian sans-serif (pricing, product specs, cart)

This creates a **two-channel system**: emotional communication lives in the serif, transactional communication lives in the sans.

## The "Unboxing" Protocol
Physical packaging was designed as a **sequential user flow**:
1. Outer shipping box (neutral, minimal branding)
2. Inner presentation box (high-contrast typography, foil stamping)
3. Product card (care instructions, authenticity certificate)
4. Jewelry pouch (reusable, branded)

Each layer reveals information **progressively**, mirroring the digital user experience.

# 3.0 OUTCOME
The system launched in 2023 and has maintained visual coherence across:
- E-commerce platform (Shopify)
- Social media (Instagram, Pinterest)
- Print collateral (lookbooks, product cards)
- Retail partnerships (wholesale presentation kits)
