## The Specification

This site functions as a **Glass Box** implementation of the protocols and systems described in Altitudes 01 and 02. It is simultaneously a portfolio and a working case study—a demonstration of the structural integrity required to bridge high-level vision with high-fidelity execution.

## The Architecture

### Altitude Structure

The site uses a three-tier Information Architecture to organize 12+ years of enterprise depth, proprietary implementation scaffolding, and high-taste creative output:

- **Altitude 01: Platforms** — Industrial governance and enterprise-scale remediation. Durable infrastructure that keeps organizations upright.
  - Core Projects: Encore (12+ years of platform sovereignty), Aiden-Jae (luxury brand architecture), Altrueism (social impact remediation)

- **Altitude 02: Protocols** — Implementation scaffolding and context hardening. The "rails" that prevent drift.
  - Core Projects: Savepoint Protocol (semantic version control for thought), Order of the Ætherwright (symbolic operating system), Culinary Mentor (knowledge infrastructure specification)

- **Altitude 03: Artifacts** — Fabricated evidence and systemic proof. High-taste output produced by the systems above.
  - Core Projects: New City (narrative fabrication), Everyday Gold (regulatory system fabrication), Photogeography (somatic visual evidence)

### Technical Foundation

**Generator**: Jekyll 4.4.1 (Ruby-based static site generation)

**Styling**: Custom SCSS framework (manual craft; no utility class bloat)

**Hosting**: GitHub Pages with custom domain

**Metadata**: Dual-layer JSON-LD + OpenGraph (Human-readable prose + machine-readable schema for SEO Semantic Parity)

**Version Control**: Git (semantic commits via Savepoint Protocol v3.0)

**Typography**: EB Garamond (human prose), Baskerville (authority headers)

**Layout**: Swiss Grid logic with halftone somatic texture anchors

## The Role: Principal Architect

A **Principal Architect specializing in Implementation Infrastructure** is responsible for:

### Governance

Ensuring the technical and visual systems remain coherent across all stakeholders and time horizons.

In enterprise: preventing .NET Core platforms from drifting into unmaintainable legacy systems.

In creative: ensuring that the "Why" (design intent) survives the "How" (technical/production implementation).
