## Quality Standards (Consistent with Weeks 2)

### Voice Standards
✓ Every sentence: "Would you say this to a peer you respect?"
✗ No marketing language (innovative, cutting-edge, passionate, synergy, paradigm, leverage, empower, journey)
✗ No aspirational tone
✗ No unjustified jargon
✓ Concrete vocabulary (build, hold, structure, system, integrity, sovereign)
✓ First person throughout
✓ Direct and specific
✓ Materials-based thinking

### Extraction Standards
✓ Every quote is verbatim (word-for-word from EXTRACTION_MINING_RESULTS.md)
✓ Context is accurate and preserved
✓ Quotes flow naturally (not awkwardly inserted)
✓ Citations are clear (date, conversation context from mining results)
✓ Quotes illuminate thinking (not just outcomes)

### Structure Standards
✓ Follow CONTENT_ARCHITECTURE_TEMPLATE.md
✓ All 6 sections present and substantive
✓ 400-600 lines each case study (4-5x current, substantive expansion)
✓ Word count distributed per targets above
✓ Expansion is from extraction content, not padding

### Build Standards
✓ Markdown is valid
✓ Frontmatter preserved
✓ No build errors when jekyll serve runs
✓ Site renders correctly locally
