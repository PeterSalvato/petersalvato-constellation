```javascript
const canvas = document.getElementById('gameCanvas');
const ctx = canvas.getContext('2d');

// Define the player as a simple geometric shape
const player = {
  x: 100,
  y: 100,
  size: 20,
  color: '#FF0000' // Primary red for the player
};

// Array to hold enemy shapes
const enemies = [
  { x: 300, y: 50, width: 40, height: 10, color: '#000' }, // black rectangles
  { x: 50, y: 300, width: 20, height: 40, color: '#000' }
];

// Function to draw the player
function drawPlayer() {
  ctx.fillStyle = player.color;
  ctx.fillRect(player.x, player.y, player.size, player.size); // Use a square for Bauhaus simplicity
}

// Function to draw enemies
function drawEnemies() {
  enemies.forEach(enemy => {
    ctx.fillStyle = enemy.color;
    ctx.fillRect(enemy.x, enemy.y, enemy.width, enemy.height);
  });
}

// Basic game loop
function gameLoop() {
  ctx.clearRect(0, 0, canvas.width, canvas.height); // Clear canvas for smooth motion
  drawPlayer();
  drawEnemies();
  requestAnimationFrame(gameLoop);
}

gameLoop();
```

## 📌 What Is a Game Developer?

A **Game Developer** is someone who makes video games. That includes the people who:
- Write the **code** that makes the game work
- Design the **levels** and how the player moves through them
- Draw the **characters**, **worlds**, and **animations**
- Create the **music and sound effects**
- Test the game to fix bugs and make it fun

Game development is a **team effort**, but many developers start small — by making their own games solo or with friends. Some go on to work for companies like **Nintendo, Epic Games, Blizzard**, or **indie studios**. Others build browser games, mobile apps, or even work on **educational, medical, or training games**.

---

## 🌍 What Types of Games Can You Build?

Game development doesn’t just mean PlayStation or Xbox games. There are **many paths** — and lots of them are accessible right now, even with a home computer.

### 💻 **Web Games**
- Built with **HTML**, **CSS**, **JavaScript**, or **TypeScript**
- Can be played instantly in a browser — no installs!
- Popular tools: **Phaser.js**, **P5.js**, **Construct**, **Bitsy**

### 🖥️ **Desktop Games**
- Made with powerful engines like **Unity**, **Godot**, or **Unreal Engine**
- Often written in **C#** (Unity), **GDScript** or **C++** (Godot), or **Blueprints/C++** (Unreal)
- Can be published on **Steam**, **itch.io**, or even sent to friends

### 🎮 **Console Games**
- Harder to get started with, but still reachable with tools like **Unity** or **GameMaker**
- Developers must get special permission to publish on **Nintendo Switch**, **PlayStation**, etc.
- Many indie games started small and grew to hit consoles (e.g., *Celeste*, *Undertale*)
